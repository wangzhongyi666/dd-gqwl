河北泽众人力资源服务有限公司

河北泽众人力资源服务有限公司（简称泽众人力）成立于2011年5月5日，公司位于石家庄市区，公司多年致力于企业及个人社会保险缴纳及基础人力资源服务提供，专注于社保细分领域创新服务及产品研发。根据目前灵活就业人员基本情况及灵活用工现象普遍，我们研创出本土专业且唯一的灵活就业参保服务平台——365灵活通，365灵活通是一款利用互联网手机为载体，实现灵活就业人员参保、缴费一体的一站式服务平台，并解决了灵活就业人员参保程序复杂繁琐，缴费时间集中且银行排队等痛点，方便、安全、快捷的服务广大灵活就业参保人员，真正做到了手机在手，参保缴费不愁的灵活就业参保服务新模式。

2018年5月，河北乐聘人力资源服务有限公司（兼职乐）入股河北泽众人力资源服务有限公司，并提供强大的系统开发团队助力365灵活保系统研发及产品迭代。使得365灵活保未来市场推广及产品开发升级有了团队及技术保障，未来前景更加美好辉煌。
